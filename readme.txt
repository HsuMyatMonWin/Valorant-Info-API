A WordPress plugin to display Valorant agent and weapon information from the Valorant API.

== Description ==

Valorant Info Plugin is a simple WordPress plugin that allows you to display Valorant agent and weapon data on your website. It fetches data from the Valorant API and provides shortcodes to display this information.

Features:
- Display agent information including name, background, and abilities.
- Display weapon information including name, category, magazine size, fire rate, reload speed, and damage.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/valorant-info` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcode `[valorant_agent_data]` to display agent information and `[valorant_weapon_data]` to display weapon information on any post or page.

== Frequently Asked Questions ==

= How do I display Valorant agent data? =

To display agent data, use the shortcode `[valorant_agent_data]` on any post or page.

= How do I display Valorant weapon data? =

To display weapon data, use the shortcode `[valorant_weapon_data]` on any post or page.

== How do I show/hide specific details in the display? ==

The Settings Page is located at the left side menu of the admin panel. Select which agent and weapon details to display.
